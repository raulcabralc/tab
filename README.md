# TAB. — Gestão Inteligente para Restaurantes 🍻

> **Status do Projeto:** Em desenvolvimento ativo

O **TAB** é um ecossistema Full Stack desenvolvido para transformar a gestão de bares e restaurantes. O foco central não é apenas o controle de pedidos, mas a entrega de uma experiência de usuário (UX) fluida, minimalista e de alta performance. O objetivo é eliminar o atrito entre a tecnologia e o ritmo acelerado de um estabelecimento real.

---

## 🛠 Tech Stack

### Frontend
- **React.js** com **TypeScript**
- **Framer Motion:** Micro-interações e transições suaves (uso de `popLayout`, `spring` e animações de layout).
- **Styled Components:** Design System proprietário baseado na estética **Rebound Dark Minimalist**.
- **Lucide React:** Iconografia moderna e consistente.
- **Axios:** Integração robusta com a API.

### Backend
- **NestJS:** Arquitetura modular, escalável e de fácil manutenção.
- **TypeScript:** Tipagem rigorosa em todo o fluxo de dados.
- **Resend:** Sistema inteligente de envio de e-mails para autenticação e recuperação.
- **PostgreSQL / MongoDB:** Persistência de dados segura e eficiente.
- **JWT:** Autenticação protegida com fluxos de Primeiro Acesso e recuperação de senha.

---

## ✨ Funcionalidades em Destaque

- **Fluxo de Autenticação "Smooth":** Sistema de recuperação de senha via código de 6 dígitos enviado por e-mail, mantendo o usuário no fluxo da aplicação e garantindo agilidade.
- **Dashboard Dinâmico:** Visualização em tempo real de faturamento, ticket médio e status da cozinha, utilizando *Skeleton Loading* para uma percepção de velocidade aprimorada.
- **Gestão do Estabelecimento:**
    - Configuração de horários de funcionamento com lógica de replicação automática para a semana.
    - Sistema de tags animadas para categorias de culinária.
    - Controle de ocupação de mesas com feedbacks visuais dinâmicos.
- **Segurança por Níveis (RBAC):** Controle de acesso baseado em funções (Admin, Gerente, Garçom).

---

## 🎨 Filosofia de Design

O projeto prioriza a clareza e a rapidez visual:
1. **Feedback Visual:** Uso extensivo de modais de status e animações de transição para guiar o usuário.
2. **Eficiência de Input:** Seletores inteligentes e máscaras de campo para reduzir erros e o tempo de preenchimento.
3. **Minimalismo:** Interfaces limpas que destacam apenas as informações cruciais para a operação do restaurante.
